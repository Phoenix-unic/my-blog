Calcilator with PyQt6

Another GUI project, this time a simple calculator with some additional functions, created with QtDesigner, Python, and the PyQt6 library. Before I started it, it didn't seem challenging at all, but during the process, I realized that some tasks were very interesting and required some additional skills.

If You are interested, here is the link to GitHub repo with this project

I'm going to show you the simple calculator python script, and than you'll see the difference with GUI
Default script

Here is a main function that, in one very simple step with eval(), returns the result and handles three types of errors. The user is able to infinite enter an expression to calculate following rules.

Basic needs were covered with QtDesigner. You can just put together labels, buttons, and any element you want, then compile it to a '.ui' file. After that, you can transform it to a '.py' file with a GUI interface usingPyQt.
