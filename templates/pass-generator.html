Secure password generator

At certain moment of life, we realize that passwords like '1111' or 'very.strong.password' are no longer secure. There are a lot of services that provides a strong passwords in a few clicks, but as a programmer, I decided to try tkinter and build a desktop app with my own hands.
So, here is my own, pritty simple version of a password generator.

If You are interested, here is the link to GitHub repo with this project

The problem i had

After creating an executable file and running it for the first time, I noticed that the Windows console was opening in the background of my application. I read many solutions, but none were helpful until I read the .spec file and found the 'console=True' parameter. Using this parameter fixed the issue.
